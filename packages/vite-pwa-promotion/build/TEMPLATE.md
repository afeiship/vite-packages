## installation
```shell
npm install -S @jswork/vite-pwa-promotion
```

## usage
1. import css
  ```scss
  @import "~@jswork/vite-pwa-promotion/dist/style.css";

  // or use sass
  @import "~@jswork/vite-pwa-promotion/dist/style.scss";

  // customize your styles:
  $vite-pwa-promotion-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
